Integrieren Sie das PrivacyBee-Datenschutz-Widget in WordPress, um die Einhaltung der GDPR und des Schweizer DSG zu gewährleisten. Unterstützt Builder, Mehrsprachigkeit und "Lazy Loading".

== Description ==

- **Einfache Integration**: Funktioniert mühelos mit gängigen Page Buildern wie Elementor, Oxygen Builder und Gutenberg.
- **Automatisierte Rechtskonformitäte**: Fügen Sie ganz einfach einen umfassenden Abschnitt über Datenschutzrichtlinien hinzu, der automatisch mit den neuesten rechtlichen Anforderungen aktualisiert wird.
- **Anpassbares Design**: Passen Sie das Design Ihres Datenschutzblocks an das einzigartige Erscheinungsbild Ihrer Website an..
- **Mehrsprachige Unterstützung**: Generieren und zeigen Sie Datenschutzrichtlinien in mehreren Sprachen an, um ein globales Publikum anzusprechen.
- **Regelmäßige Updates**: PrivacyBee prüft und aktualisiert regelmäßig die integrierten Dienste Ihrer Website und aktualisiert Ihre Datenschutzrichtlinie entsprechend.
- **Kosteneffizient und benutzerfreundlich**: Erstellen Sie schnell rechtskonforme Datenschutzrichtlinien, ohne dass Sie umfangreiche juristische Kenntnisse oder hohe Anwaltskosten benötigen.

== 3rd Party Services ==

Dieses Plugin verlässt sich auf den Drittanbieterdienst **PrivacyBee**, um Datenschutzrichtlinien zu verwalten und die Einhaltung der GDPR, des Schweizer DSG und anderer Datenschutzbestimmungen sicherzustellen.

Bitte vergewissern Sie sich, dass Sie über die erforderlichen Genehmigungen zur Nutzung dieses Dienstes verfügen und dass Sie die geltenden Nutzungsbedingungen einhalten.

Der Dienst von PrivacyBee wird für die automatische Erstellung und Aktualisierung von Datenschutzrichtlinien sowie für die Verwaltung von Cookie-Zustimmungen verwendet. Dieser Dienst ist erforderlich, um die Einhaltung der gesetzlichen Bestimmungen zu gewährleisten, und die Daten können in Übereinstimmung mit den Datenschutzbestimmungen von PrivacyBee weitergegeben werden. Weitere Informationen über den Umgang mit Daten finden Sie in den Nutzungsbedingungen und der Datenschutzrichtlinie von PrivacyBee.
== Installation ==

1. **Das Plugin hochladen**:
   - Gehen Sie zum WordPress Admin Dashboard.
   - Navigieren Sie zu `Plugins > Neu hinzufügen > Plugin hochladen`.
   - Wählen Sie die Datei "privacybee.zip" und klicken Sie auf "Jetzt installieren".

2. **Aktivieren Sie das Plugin**:
   - Klicken Sie nach der Installation auf "Aktivieren", um das Plugin zu aktivieren.

3. **Das Plugin konfigurieren**:
   - Gehen Sie zum Menü "PrivacyBee" in der WordPress-Admin-Seitenleiste.
   - Geben Sie Ihren API-Schlüssel ein, wählen Sie den rechtlichen Rahmen und die Sprache aus und konfigurieren Sie die Sichtbarkeit der Kopfzeile nach Bedarf.


== Usage ==

1. **Hinzufügen des PrivacyBee-Widgets**:
   - Verwenden Sie WPBakery, Gutenberg oder Elementor, um das PrivacyBee-Widget zu Ihrer Seite hinzuzufügen.
   - Konfigurieren Sie die Einstellungen des Widgets nach Ihren Wünschen.

2. **Einstellungen anpassen**:
   - Navigieren Sie zum Menü "PrivacyBee" in der WordPress-Admin-Seitenleiste, um die Einstellungen des Plugins anzupassen.

Nach der Installation fügt PrivacyBee eine neue Blockoption in das Verzeichnis Ihres Page Builders ein, mit der Sie mühelos einen vollständig anpassbaren Abschnitt über Datenschutzrichtlinien einfügen können. Ganz gleich, ob Sie eine neue Website erstellen oder eine bestehende aktualisieren, PrivacyBee bietet Ihnen die Werkzeuge, um sicherzustellen, dass Ihre Website rechtskonform bleibt, was Ihnen Zeit spart und Risiken verringert.

== Why Choose PrivacyBee? ==

PrivacyBee bietet eine einfache und effiziente Möglichkeit, eine rechtskonforme Datenschutzrichtlinie für Ihre Website zu erstellen. Mit seiner einfachen Integration, anpassbaren Optionen und regelmäßigen Updates hilft PrivacyBee Ihrer Website, die neuesten rechtlichen Standards zu erfüllen, damit Sie sich auf das Wachstum Ihres Unternehmens konzentrieren können.

Laden Sie PrivacyBee noch heute herunter und stellen Sie sicher, dass die Datenschutzrichtlinien Ihrer Website immer mit den neuesten gesetzlichen Anforderungen übereinstimmen!

== Multilingual Support ==

Die Datenschutzerklärung ist in Englisch, Deutsch, Französisch und Italienisch verfügbar. Das Plugin selbst unterstützt derzeit Englisch und Deutsch. Zukünftige Updates werden französische und niederländische Übersetzungen enthalten. Sie können Übersetzungen mit den Standardmechanismen von WordPress verwalten.

== Troubleshooting ==

- **Widget wird nicht angezeigt**: Stellen Sie sicher, dass der API-Schlüssel korrekt konfiguriert ist und keine Konflikte mit anderen Plugins oder Themes bestehen.
- **Performance-Probleme**: Überprüfen Sie, ob das Skript `widget.js` korrekt geladen wird und ob es keine Probleme mit der PrivacyBee API gibt.
